# simple_product_viewer_app


## Setup Instructions

### 1. Install FVM & Flutter

Refer to the [FVM installation guide](https://fvm.app/documentation/getting-started/installation).

### 2. Install dependencies


```bash
fvm flutter pub get
```

### 3. Run the app

```bash
fvm flutter run
```
